Project Assignment



For your project you will create an R markdown file (ends with the file ending .Rmd). Your R markdown file must:
1. An appropriate header section with your name and date
2. Contain a description of the data and what it is that you wish to test in the data written as plain text outside of code blocks
3. Read a CSV dataset from the same directory as it is in (do not use setwd, or provide a file path when reading data in)
4. A statistical test of some aspect of your data
5. An interpretation of the results of your statistical test in plain text outsdie of code blocks
6. A high quality visualization of your data and or findings.
7. A plain text description of your plot as would be appropriate for a figure legend in a submitted manuscript


Your project will be submitted as two files 1) an R markdown file and 2) a CSV file containing your data if you choose to not use one of the datasets below. If you use one of the datasets below do not submit a copy of the dataset (I have it) and do not change the file name of the datafile when you download it.



Dataset 1: This dataset contains horn sizes for 107 beetles. These beetles have horns that are used in male/male combat. In general horn size varies with body size howerver some beetles have alleles that lead to larger horns and some have alleles that lead to smaller horns. If you want to artificially select on horn size you need to identify the beetles that should be bred to increase or decrease horn size independent of body size. The goal with this dataset is to use what you have learned over the course of the class to identify the 10 beetles that should be used to breed for large horn size and the 10 beetles that should be used to breed for small horn size.
gantocerus.csv



Dataset 2: This dataset contains mass, mating, and offspring number information for 48 salamanders. The goal with this dataset is to see if either mass or number of matings have an impact on the number of offspring produced. Use what you have learned including posthoc tests to determine how these possible predictor variables impact offspring number. Note it is not sufficient simply to say that a predictor does impact offspring number discuss how it impacts it and what treatments are the same or different from one another.
newts.male.csv


Dataset 3: This dataset contains 10 morphological measurements for 200 frog specimens. 190 of these values are associated with specimens that have been identified to a specific species. However, 10 of the samples are from specimens that have not been confidently assigned to a species. Use what you have learned to classify as many of these 10 samples to one of the species included in the dataset.
amphibia.csv
