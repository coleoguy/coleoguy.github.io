Teaching Resources


I believe that R is currently an essential tool for most evolutionary biologists. I built a seminar course to teach graduate students the basics of working in R. Each week had a tutorial covering a core set of essential R skills that the participants learned. This website contains all of the tutorials and associated datafiles required for the course.



An understanding of the foundational ideas that the modern synthesis of evolutionary biology is built on is essential for asking intelligent questions and understanding how your research fits into a larger understanding of life. I offered a seminar course for graduate students that provided a structured environment to discuss some of the most important papers in evolutionary biology. This website contains a schedule and PDF files for all of the readings in this course.



Wright-Fisher simulators perform population genetic simulations that include both selection and drift. Interactive simulators like this one can be particularly helpful in teaching students about the impact that population size has on the efficiency of selection. This website uses the R package shiny to create an interactive simulation that allows users to control mutation rates, selection coefficients, and population sizes.



Many bayesian phylogenetics programs require specification of statistical distributions to describe the prior probability of parameters. This website allows users to select a statistical distribution and then vary the parameter values that specify its shape.



A birth-death model is often assumed as a generating model for phylogenies. One challenge in detecting shifts in diversification is the high variance in trees generated under this model. This website allows users to set values for the birth and death parameters and then generates a collection of trees to illustrate the high variance that is often exhibited.
